Alderbridge — shared inventory model 共享库存说明
Alderbridge FOH and CS share the same Bento inventory pool. CS handles all online/phone reservations and label posting for those orders. FOH handles walk-in bookings and label posting for those.
A店库存由客服与前台共用。客服负责线上/电话预订及贴单；前台负责 Walk-in 现场预订及贴单。

Who triggers Hold Stock replenishment?
CS reviews inventory across all 3 stores after closing each day. If any store is running low, CS arranges the replenishment and notifies both the store and Stacy. FOH staff do not request Hold Stock independently.

Where to post the label? 贴单位置判断

Inventory cycle definition: Each cycle runs from Thursday (restock day) through the following Wednesday. Any pickup date that falls within the same Thu–Wed window as the order date is "within this cycle".
库存周期定义：每个周期从周四出货日开始，至下周周三结束（Thu–Wed）。预取日在同一周期内即为"本次周期内"。

Why system On Hand ≠ physical count
System On Hand = Physical available boxes − Labels posted outside freezer
So when entering stock received: On Hand to enter = System On Hand + qty received
系统 On Hand = 实际可卖盒数 − 冰箱外贴单数，两者存在差别。
因此录入时：填入数值 = 当前系统 On Hand + 实际收货数
Example 举例：
System shows −2 · Receiving 25 boxes → enter On Hand = −2 + 25 = 23
Then check green number shows +25 = boxes actually received ✓

Cancelling a paid reservation 取消已付款预订

Cancelling a paid order restores inventory automatically — no need to manually add stock back.
取消已付款订单后系统会自动恢复库存，无需额外手动添加库存。
1
Open the order → tap "…" (top right) → tap "Cancel items" 打开订单 → 点右上角 "…" → 选择 "Cancel items"
Step 1 — Tap "Cancel items" from the "…" menu
2
Select "Customer request" or "Other" based on the situation → tap "Cancel items"

Both options restore inventory automatically. Refund decision is based on the actual situation.

根据实际情况选 "Customer request" 或 "Other" → 点 "Cancel items" 确认。退款与否视实际情况决定。
Step 2 — ✅ Select "Customer request" or "Other" (both restore stock)  ·  ❌ Do NOT select "Item(s) unavailable" — marks inventory as lost
